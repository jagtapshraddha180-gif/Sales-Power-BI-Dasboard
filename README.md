
# Sales Dashboard using Power BI 📊

## 📌 Project Overview
This project presents an interactive sales dashboard built using Power BI to analyze key business metrics such as revenue, profit, and sales trends.

## 🚀 Features
- Interactive visualizations and filters
- Sales performance tracking
- Region-wise and product-wise analysis
- KPI indicators

## 🛠 Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning
- DAX

- Sales revenue performance.pbix
